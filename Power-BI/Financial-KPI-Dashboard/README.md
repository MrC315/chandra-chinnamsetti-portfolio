## Overview

Developed a 3-page interactive Power BI dashboard to analyze financial performance, profitability, and transaction-level insights.

## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling

## Dashboard Pages

### Executive Financial Dashboard
Provides executive-level KPIs including Revenue, Profit, Gross Sales, Units Sold, Profit Margin %, and Discount %.

### Profitability Analysis
Analyzes product profitability, discount impact, segment performance, and top/bottom products.

### Financial Detail Explorer
Provides drill-through capability for detailed analysis and transaction-level reporting.

## Key Skills Demonstrated

- Data Modeling
- Advanced DAX
- Dashboard Design
- Financial Analytics
- Drill Through Navigation
- Conditional Formatting
- Interactive Reporting
